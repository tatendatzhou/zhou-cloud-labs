# Project 1 — Azure Resource Groups (Portal)

## Objective
Learn how to create and manage Azure Resource Groups using the Azure Portal.

Resource Groups are logical containers used to organize and manage Azure resources such as Virtual Machines, Storage Accounts, and Networking components.

---

## Steps Performed

1. Logged into the Azure Portal
2. Navigated to **Resource Groups**
3. Clicked **Create**
4. Entered the following values:

Resource Group Name:
RG-LAB1

Region:
UK South

5. Clicked **Review + Create**
6. Clicked **Create**

---

## Result

The Resource Group was successfully created and can now be used to deploy Azure resources.

Example properties:

| Property | Value |
|--------|------|
| Resource Group | RG-LAB1 |
| Location | UK South |
| Provisioning State | Succeeded |

---

## What I Learned

- Resource Groups are used to logically organize Azure resources
- Azure resources must belong to a Resource Group
- The Azure Portal provides a graphical interface for managing resources
