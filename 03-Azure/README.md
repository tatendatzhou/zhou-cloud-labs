# Azure Administration Labs (AZ-104)

This repository contains hands-on Azure infrastructure labs used to reinforce and demonstrate practical skills covered in the **Microsoft Azure Administrator (AZ-104)** certification.

The labs document how Azure resources are deployed and managed using **both the Azure Portal and Azure PowerShell**.
Each lab includes the manual configuration steps as well as the equivalent PowerShell automation.

---

## Repository Structure

Each folder focuses on a specific Azure resource or administrative task.

```
azure-az104-labs
│
├── 01-Resource-Groups
├── 02-Storage-Accounts
├── 03-Virtual-Networks
├── 04-Network-Security-Groups
└── 05-Virtual-Machine
```

Each lab contains two sections:

**Portal**
Documentation and screenshots showing how the resource was created using the Azure Portal.

**PowerShell**
PowerShell scripts that automate the same deployment.
